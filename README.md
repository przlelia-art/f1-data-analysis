# Analyse des données F1 - GP Bahreïn 2024 📊

Ce projet est une analyse des données de Max Verstappen et Charles Leclerc lors du Grand Prix de Bahreïn 2024, réalisée par passion pour la Formule 1 et pour apprendre la data analyse avec Python.

## 🏎️ Analyses disponibles

### GP de Bahreïn 2024

- **Comparaison des temps au tour** - Évolution des performances des deux pilotes tout au long de la course.
- **Stratégie des pneus** - Visualisation des composés utilisés (SOFT/HARD) et leur impact sur les temps au tour.
- **Analyse de la télémétrie en qualifications** - Comparaison de la vitesse de Verstappen et Leclerc mètre par mètre sur leur tour le plus rapide.

### GP de Monza 2024

- **Comparaison de vitesses maximales** - Analyse des vitesses sur ligne droite de Verstappen, Leclerc et Norris

## 🛠️ Technologies utilisées

- Python
- FastF1
- Pandas
- Matplotlib
- Seaborn
 
## 🚀 Comment lancer le projet

```bash
# Cloner le repo
git clone git@github.com:przlelia-art/f1-data-analysis.git
 
# Activer l'environnement virtuel
source venv/bin/activate
 
# Installer les dépendances
pip install fastf1 pandas matplotlib seaborn
 
# Lancer une analyse
python3 analyse_f1.py
python3 strategies_pneus.py
```
